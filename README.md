## 📋 Overview

AI Career Coach is a comprehensive web platform that leverages artificial intelligence to provide:

- **Resume Analysis** - AI-powered scoring with actionable improvement suggestions
- **Job Role Targeting** - Role-specific resume optimization
- **Interview Practice** - Simulated interviews with AI-generated questions and feedback
- **Progress Tracking** - Visual dashboards to monitor career development

## 🛠️ Technology Stack

- **Backend**: Java 25, Spring Boot 3.2.4, Spring Security, JPA/Hibernate
- **Database**: PostgreSQL 16
- **AI**: OpenAI API (GPT-4o)
- **Frontend**: React.js 18 + Tailwind CSS (coming soon)
- **Authentication**: JWT (JSON Web Tokens)
- **Build Tool**: Maven
- **Version Control**: Git + GitHub
